## Project Overview
This repository contains **UI automation tests** for a To-Do List web application using **Robot Framework** and **SeleniumLibrary**.

The test suite focuses on validating core user workflows such as:
- Adding tasks
- Verifying tasks list
- Deleting tasks
- Completing tasks
- Deleting Completing tasks
- End-to-End (E2E) task management flow

All test cases were executed successfully with **100% pass rate**.
